自动保存工具

自动自动保存工具自动保存和清除记录会话在设置的时间间隔

如果您让Charles长时间监视网络活动，这是很有用的 并希望将记录分解成易于管理的单元，或避免内存不足的情况 这可能是由于大量数据造成的。

输入保存间隔(以分钟为单位)和保存会话文件的目录。 您可以选择每次运行Charles时是否启动自动保存工具，否则 当Charles启动时，自动保存工具将永远被禁用。

您可以选择启动 保存间隔计时器上的保存间隔分钟过去一小时的倍数。您的第一个保存间隔将 使其更短，以对齐间隔到倍数。例如，如果您的保存间隔是15分钟 您在10:10启动工具，第一个保存间隔将是10:15，以对齐15分钟的倍数， 之后每15分钟就会有一次。

会话文件的名称中带有时间戳，格式为yyyyMMddHHmm，即年、月、日、小时、分钟 这样当按字母顺序排序时，它们就会以正确的顺序出现。
