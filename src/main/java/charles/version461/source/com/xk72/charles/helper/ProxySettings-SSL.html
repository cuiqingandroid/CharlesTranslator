SSL代理

SSL代理 拦截所有SSL请求并可以SSL请求的纯文本内容，而不仅仅是查看加密的文本

开启 SSL代理 , 您的浏览器或应用程序将收到由Charles签名的证书，而不是来自远程web服务器的原始证书。 这将在浏览器或应用程序中触发警告， 有些应用程序可能会拒绝连接

您可以将您的浏览器或应用程序配置为接受Charles的证书 http://www.charlesproxy.com/ssl或在Charles的帮助菜单中的SSL代理菜单。

匹配规则

应该逐个列出需要Charles抓包SSL请求。为保护您的个人信息安全，您应该只包含您想要的抓包SSL站点。

可以将通配符添加到位置列表中，以匹配多个或所有位置。要匹配所有位置，请输入*。

对于要被代理的站点，它必须同时匹配包含位置和不匹配任何排除位置。 当您在包含位置中使用了通配符时，排除位置最有用。
