User Interface Settings

Look and Feel

The native platform look and feel is usually the best as it makes Charles appear like other applications on your system. However you can choose another look and feel to suit your preferences or requirements.

Display Font

The display font is used in some viewers such as the Text viewer. If you have specific requirements for viewing information you can change the font here, such as to a font that includes foreign-language character sets.

Display Font Size

The size to use for the display font.

Minimise to System Tray

Windows only option. When Charles is minimised its window will be hidden from the task bar and can be restored by clicking the Charles icon in your system tray.

Enable Global Hotkeys

Windows only option. Bring Charles to the front from any application by pressing either WINDOWS-C or ALT-CONTROL-SHIFT C.

Show Memory Usage

Show Charles's memory usage in the status bar.
